# README
## Ruby Demonstration Project
### Developed using:
* Ruby 3.0.2
* Rails 6.1.4
* Node 14.17.4
* Yarn 1.22.11
* SQLite3

### Initial Setup
```
cd app
bundle install
yarn install
rake db:setup
rails s
```

The site will be on [localhost](http://127.0.0.1:3000)

### Features
Adding, editing, and deleting ratings work
